My Opinion about HHVM and Hacklang

I never use both technologies but here's my perspective
after I briefly searched for the informations.
In my opinion HHVM great for manage a Website that receives a large volume of concurrent hits because
HHVM requires less memory space to execute requests.
also because Facebook uses HHVM, they are constantly working to improve it,
so developers have dependable community support.


Things should we consider before migrating

1. Because Facebook uses HHVM,
In the unlikely event that Facebook discontinues support for HHVM,
the large development and support community would cease to exist.

2. For developers who have been using PHP for years, HHVM will feel very unfamiliar,
so there's learning curve to overcome.
because developers will learn again know how to set it up, configure, and code with it.
